# CompensIQ

**ML-powered salary intelligence for smarter hiring decisions.**

🔗 [Live Demo](https://compensiq-rzhdftchaheptc3tn22dsp.streamlit.app)

---

## What it does

CompensIQ predicts salaries across **62 countries**, **7 job families**, and **7 job titles** — trained on 50,000+ records — and delivers explainable, business-ready outputs via a Streamlit web app.

---

## Results

| Metric | Value |

